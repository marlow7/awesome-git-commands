# 关于 awesome-git-commands

## 项目简介

awesome-git-commands 是一个基于 Git 官方原生文档打造的、以暗色主题为默认的 Git 命令全量可视化 Web 站点。

本项目旨在解决以下痛点：

- Git 官方文档无原生暗色模式
- 官方分类不贴合日常开发工作流
- 实战示例缺失，新手难以快速上手
- 移动端适配差

## 内容来源声明

::: important 核心声明
本项目核心内容来源于 **Git 官方文档**（https://git-scm.com/docs/git），遵循 **GPLv2 开源协议**。

- 所有 Git 命令的定义、语法格式、参数说明、功能描述均对齐 Git 官方文档
- 官方原生内容已明确标注来源，不得声称官方内容为项目原创
- 项目原创补充内容（实战示例、指南、踩坑说明）已与官方原生内容明确区分
- 本项目非 Git 官方出品，仅作学习与使用参考
:::

## 开源协议

本项目遵循 **GNU General Public License v2.0 (GPLv2)** 开源协议，与 Git 官方文档的开源协议保持一致。

## 项目价值

| 特性 | 说明 |
|------|------|
| 官方权威内容 | 所有命令定义100%对齐Git官方文档 |
| 默认暗色主题 | 优先保障暗色模式视觉舒适度 |
| 全量命令覆盖 | 覆盖Git全量命令，包含底层命令 |
| 实战示例补充 | 每个核心命令补充可直接使用的实战示例 |
| 极速全局搜索 | 全局模糊搜索，响应延迟低于200ms |
| 全终端响应式 | 桌面端、平板端、移动端全适配 |
| 完全开源免费 | GPLv2协议，免费使用 |

## 技术栈

- **站点生成**：VitePress 1.x
- **代码高亮**：Shiki（VitePress 内置）
- **全局搜索**：MiniSearch（VitePress 内置）
