# 分支管理与切换

创建、切换、删除分支等分支管理操作，是多人协作和功能开发的核心能力。

<a class="agc-cmd-list-item" href="/commands/scene-category/branch-management/git-branch">

  </div>
  <span class="cmd-desc">列出、创建或删除分支</span>
</a>
