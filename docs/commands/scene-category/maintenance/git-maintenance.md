# git maintenance

在后台运行任务以优化 Git 仓库数据。

<a class="official-link" href="https://git-scm.com/docs/git-maintenance" target="_blank" rel="noopener noreferrer">Git 官方文档 → git-maintenance</a>

## 实战示例

```bash
# 启用自动维护
git maintenance start

# 停止自动维护
git maintenance stop

# 手动运行维护任务
git maintenance run
```
