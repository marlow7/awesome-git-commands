# 第三方交互命令（Interacting with Others）

> 官方定义：与外部系统交互的命令，主要用于通过邮件等非 Git 原生协议进行协作。
>
> 内容来源：[Git 官方文档 - Interacting with Others](https://git-scm.com/docs/git#_interacting_with_others)

## 命令列表

| 命令 | 官方描述 | 标签 |
|------|----------|------|
| git archimport | Import a GNU Arch repository into Git | <span class="agc-tag agc-tag-low">底层</span> |
| git cvsexportcommit | Export a single commit to a CVS checkout | <span class="agc-tag agc-tag-deprecated">废弃</span> |
| git cvsimport | Salvage your data out of another SCM people love to hate | <span class="agc-tag agc-tag-deprecated">废弃</span> |
| git cvsserver | A CVS server emulator for Git | <span class="agc-tag agc-tag-deprecated">废弃</span> |
| git imap-send | Send a collection of patches from stdin to an IMAP folder | <span class="agc-tag agc-tag-low">底层</span> |
| git p4 | Import from and submit to Perforce repositories | <span class="agc-tag agc-tag-low">底层</span> |
| git quiltimport | Applies a quilt patchset onto the current branch | <span class="agc-tag agc-tag-low">底层</span> |
| git request-pull | Generates a summary of pending changes | <span class="agc-tag agc-tag-common">常用</span> |
| git send-email | Send a collection of patches as emails | <span class="agc-tag agc-tag-common">常用</span> |
| git svn | Bidirectional operation between a Subversion repository and Git | <span class="agc-tag agc-tag-deprecated">废弃</span> |
