# Git Command Chinese Reference Manual

A comprehensive Chinese Git reference strictly following the official documentation. <br>It covers core commands from basic to advanced, including definitions, syntax, parameters, practical examples and common pitfalls.

</div>


## Why This Repo
- **Authoritative & Accurate**: Content strictly follows the official Git documentation to ensure professional, reliable, and up-to-date information.
- **Chinese-Friendly**: Fully explained in Chinese with no language barriers, perfect for Chinese-speaking developers.
- **Practical-Oriented**: Each command provides the most commonly used examples that you can copy and use directly.
- **Well-Structured**: Commands are categorized by usage scenarios for quick lookup and learning.


## Quick Start
### Use Online
Visit the website to browse all Git command documentation:
[https://marlow7.github.io/awesome-git-commands/](https://marlow7.github.io/awesome-git-commands/)


### Local Development
```bash
# Install dependencies
pnpm install

# Start local development server
pnpm dev

# Build for production
pnpm build
```
